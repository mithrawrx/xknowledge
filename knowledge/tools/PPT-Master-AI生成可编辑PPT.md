AI 生成原生可编辑 PPT，每个元素都是真实的 PowerPoint 形状——可点击、可编辑、可直接发给老板。

## 核心特点

- **原生可编辑**: 输出的每个元素都是真实的 PowerPoint 形状，不是图片或网页截图
- **多格式输入**: 支持 PDF、DOCX、Markdown 文件
- **本地处理**: 配合 Claude Code 使用，文件不上传服务器
- **开源免费**: 5,780+ stars

## 使用方法

1. 下载项目：`git clone https://github.com/hugohe3/ppt-master`
2. 安装依赖：`pip install -r requirements.txt`
3. 把 PDF、DOCX、Markdown 丢进 `projects/` 目录
4. 告诉 AI："请用这个文件生成 PPT"
5. 导出文件在 `exports/` 目录

## 背景

由金融从业者 Hugo He 开发，他每天审阅数百张 PPT，无法忍受"导出图片"的方案，于是自己写了这个工具。
