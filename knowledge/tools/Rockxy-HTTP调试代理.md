# Rockxy - HTTP调试代理

开源原生 macOS HTTP(s) 调试代理工具。

## 核心功能

- **拦截 HTTPS**：完整捕获 HTTPS 请求
- **API 分析**：深入分析 API 调用细节
- **修改请求**：实时修改请求内容
- **重放功能**：支持请求重放测试
- **HAR 导出**：导出 HAR 格式用于分享和分析

## 技术特点

- Swift 原生开发，无 Electron 依赖
- 性能强劲，资源占用低
- 无云依赖，数据本地处理
- 开源可审计

## 适用场景

- 开发者日常 API 调试
- 接口问题排查
- 网络请求分析
