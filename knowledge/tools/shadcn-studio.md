Shadcn Studio 是开源的 shadcn/ui 组件、区块和模板集合，配以强大的主题生成器。扩展了核心 shadcn/ui，提供更多样化的组件变体、预建区块和完整模板，加速 UI 开发。

与传统 npm 安装的库不同，shadcn/studio 沿用 shadcn/ui 的"复制粘贴"哲学，让开发者直接在项目中拥有和定制所有代码，无库依赖束缚。

## 核心功能

- **25+ 日历组件变体** - 单选/范围选择、多月导航、时间段、预设
- **丰富组件变体** - 比核心 shadcn/ui 更多变体
- **预建区块** - 即用的 UI 区块和模式
- **落地页和仪表板** - 20+ 高级和免费模板
- **主题生成器** - 实时定制和预览
- **Shadcn/ui for Figma** - 包含组件、区块、模板的设计库
- **MCP 集成** - 直接集成到 IDE 生成组件
- **Figma 转 Code 插件** - 将 Figma 设计转为代码
- **开源** - MIT 许可证，社区驱动

## 技术细节

- **语言:** TypeScript
- **样式:** TailwindCSS
- **仓库:** github.com/themeselection/shadcn-studio
- **Star 数:** 950+
- **分发模式:** 复制粘贴（非 npm）

## 使用场景

- 构建无组件库束缚的自定义 UI 系统
- 使用预设计变体快速原型开发
- 创建一致的主题化应用
- Figma 转 Code 工作流
