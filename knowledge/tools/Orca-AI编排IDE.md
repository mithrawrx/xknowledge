Orca 是下一代 IDE，专为使用 coding agents 构建 100x 效率而设计。可以在一个界面中同时运行 Claude Code、Codex、OpenCode 等多个 AI agent，每个 agent 在独立的 worktree 中运行，统一追踪管理。

## 核心功能

- **多Agent并行运行**: 支持 Claude Code、Codex、Gemini、Pi、OpenCode、Goose、Amp、Auggie、Cline 等数十种 CLI agent
- **Worktree 管理**: 每个 agent 在独立的 git worktree 中运行，避免冲突
- **内置Ghostty终端**: 集成高性能终端
- **文件编辑器**: 右侧文件目录，可直接查看文件内容
- **跨平台**: 支持 macOS、Windows、Linux

## 解决的问题

- 解决同时打开多个 Claude Code 命令行窗口难以管理的问题
- 提供可视化文件浏览，方便将文件路径告知 agent
- 执行完成后高亮通知
