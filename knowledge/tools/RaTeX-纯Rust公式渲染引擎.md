RaTeX 是纯 Rust 编写的 LaTeX 数学公式渲染引擎，KaTeX 语法覆盖率超过 99.5%。

## 核心功能

- **>99.5% KaTeX 覆盖**: 几乎完整支持 KaTeX 语法
- **纯 Rust 实现**: 无 JS 依赖，高性能编译
- **数学公式渲染**: LaTeX 数学表达式到高质量渲染输出
- **教育 App 友好**: 适合集成到教育类应用中作为公式渲染引擎

## 技术优势

- Rust 原生实现，编译速度快、运行时性能高
- 相比 JS 方案（KaTeX/MathJax），可作为 WASM 或原生库集成
- 对教育 App 的技术选型有颠覆性潜力
