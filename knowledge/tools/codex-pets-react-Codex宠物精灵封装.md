为 Codex 宠物精灵 spritesheets 提供声明式 React 封装，内建 8 款复古风格像素宠物动画。

## 核心功能

- **声明式 API**: React 组件方式直接调用宠物精灵动画
- **内建 8 款宠物**: 复古像素风格，动画效果流畅有感觉
- **spritesheet 封装**: 自动处理精灵帧动画渲染
- **零配置**: 安装即用，无需额外设置
