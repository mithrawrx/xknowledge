原生 macOS LLM 服务器，支持 MCP。Osaurus 是一个在 Apple Silicon 上运行语言模型的全能平台，提供 OpenAI 和 Anthropic 兼容 API、MCP 服务器能力，以及内置插件生态系统。

让 Claude 和其他 AI 智能体能看到你的屏幕、点击、输入和导航——完美展示现代 macOS 系统上 AI 智能体的形态。结合本地 MLX 推理，支持远程服务商（Anthropic、OpenAI、OpenRouter 等）。

## 核心功能

- MLX 运行时，优化 Apple Silicon 推理
- OpenAI、Anthropic 和 Ollama 兼容 API
- MCP 服务器，集成 Cursor、Claude Desktop
- 多窗口对话，自定义角色
- 语音输入，WhisperKit 转录
- 插件系统，可扩展
- 菜单栏对话覆盖层
- 支持 Apple Foundation Models（macOS 26+）
