微软开源的文档转 Markdown 工具，支持将 PDF、Word、PPT、Excel 等多种格式转换为 Markdown，方便 LLM 处理和知识管理。

## 核心功能

- **多格式支持** - PDF、Word、PPT、Excel、HTML 等
- **Markdown 输出** - 统一的 Markdown 格式输出
- **LLM 友好** - 方便大语言模型处理文档内容
- **批量转换** - 支持批量处理文档

## 技术细节

- **仓库:** github.com/microsoft/markitdown
- **Star 数:** 95,815

## 使用场景

- 文档知识库构建
- LLM 文档处理
- 内容迁移和格式转换
- 自动化文档处理流水线
