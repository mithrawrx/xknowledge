Claude Island is a native macOS application written in Swift that provides Claude Code notifications via the Dynamic Island (the notch area on modern Macs). It displays an expandable overlay in the Dynamic Island, allowing users to monitor Claude Code activity without context switching.

This tool is particularly valuable for developers who use Claude Code extensively and want to stay informed about background tasks, completions, and status updates without interrupting their current workflow.

## Key Features

- **Dynamic Island Integration** - Native macOS Dynamic Island support for notifications
- **Swift Native App** - Built entirely in Swift for optimal macOS performance
- **Expandable Overlay** - Click to expand notification details from the notch
- **No Context Switching** - Monitor Claude Code without leaving your current application
- **Real-time Status** - Track Claude Code activity and completions
- **Lightweight** - Minimal system resource usage

## Architecture

The app integrates with Claude Code's notification system:
1. Listens for Claude Code events and status changes
2. Displays notifications in the Dynamic Island area
3. Provides expandable UI for detailed information
4. Supports multiple concurrent notifications
