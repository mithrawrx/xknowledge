Career-Ops 是一个基于 Claude Code 构建的 AI 驱动求职系统，支持 14 种技能模式、Go 语言仪表板、PDF 生成和批处理功能。该项目在开源后迅速获得关注，Star 数超过 26,000。

## 核心功能

- **14 种技能模式** - 多种求职相关功能模块
- **Go 仪表板** - 使用 Go 构建的管理界面
- **PDF 生成** - 自动生成求职相关文档
- **批处理** - 支持批量处理求职申请
- **Claude Code 集成** - 深度集成 Claude Code 能力

## 技术细节

- **语言:** JavaScript/TypeScript
- **平台:** Claude Code 扩展
- **仓库:** github.com/santifer/career-ops
- **Star 数:** 26,482+
- **使用场景:** AI 辅助求职、简历生成、求职申请自动化

## 使用场景

- 大规模求职申请自动化
- AI 驱动的简历优化
- 求职流程管理
- 批量申请处理
