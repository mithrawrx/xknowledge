# iamzhihuix/skills-manage

用了两天志辉大佬开源的 skills-manage，好家伙，这个工具解决了我一个大痛点。

以前我的 Skill 文件散落在各种地方，每次更新一个 Skill，得手动同步。

skills-manage 用 ~/.agents/skills/ 做中央仓库，其他平台通过软链接指向这里。一处修改，全平台生效。
