# zilliztech/claude-context

今日 GitHub Trending 榜首，Claude Context 这个开源项目专门解决 AI 编程工具的上下文窗口不够用的问题。

通过 MCP 插件，让 AI 编程助手拥有对整个代码库的语义搜索能力，精准找到相关代码。

向 AI 提问时只检索相关代码片段，不用把整个项目塞进上下文。

在检索质量相当的前提下，据测试能减少约 40% 的 token 消耗。
