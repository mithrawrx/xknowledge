开源英语学习神器，用「连词成句 + 循序渐进」的方式学英语，不再死记硬背单词，而是不断重复形成肌肉记忆。融入游戏奖励和积分排行榜，把枯燥的背单词变成刷分上榜的爽感！

## 核心功能

- **连词成句学习法**: 通过构建句子学习英语，形成肌肉记忆而非死记硬背
- **游戏化机制**: 积分排行榜、奖励系统，让学习有刷分上榜的爽感
- **开源免费**: 完全开源，支持自托管
- **本地运行**: 支持本地运行，隐私安全拉满
- **跨平台**: Web 应用，支持多平台访问

## 技术栈

- TypeScript
- Node.js >= v20
- pnpm >= 8
- Postgres >= 14.0.0
- Redis >= 5.0.0
- Docker

## 使用场景

- 想高效提分的英语学习者
- 想摆脱死记硬背的学生
- 想本地跑专属英语学习站的技术爱好者
- 追求隐私安全、不想数据上传云端的自托管用户
