## 概述

本文详细分享了一个法国验光师如何从零基础学习编程，做出AI视频工具 StoryShort.ai，达到月收入20万人民币的完整攻略。核心思路是：找到已验证的产品，做一个自己的版本，快速上线。

## 核心内容

### 一、为什么值得做？

**三个优势：**
- 门槛低：不需要编程基础，用AI编程工具（Cursor + Claude Code）就能完成开发
- 市场刚起步：AI视频生成技术2025年爆发、2026年进入商业落地期
- 客单价高：视频比图片值钱，用户愿意付$20-50/月甚至更多

**赚什么钱：**
- 信息差的钱：帮用户打包好API能力，用户打开就能用
- 本地化的钱：多语言、适配本地支付渠道和交互习惯
- 场景定制的钱：针对特定类型视频做差异化功能

### 二、搭建开发环境

**电脑配置：**
- Windows 10+ 或 macOS 10.15+
- 内存至少8GB（推荐16GB）
- 硬盘预留至少10GB

**安装工具：**
- Node.js LTS版
- Git
- Cursor编辑器 + Claude Code插件
- cc-switch工具（切换底层模型降低成本）

### 三、域名与服务配置

**域名注册建议：**
- 推荐后缀：.com .net .app .io .studio .design .art .pro
- 避免：.ai（贵）、.xyz .top .vip（垃圾域名）
- 蹭词站域名要包含目标关键词

**核心服务账号：**
- Vercel（托管，免费版即可）
- Neon（数据库）
- MiniMax API（视频生成）
- Cloudflare（域名解析、R2存储）
- Resend（邮件服务）

### 四、从模板开始建站

推荐使用开源模板改造，如 VideoFly 模板。核心配置：
- 数据库初始化
- Google登录配置
- Cloudflare R2文件存储
- Resend邮件服务
- 管理员邮箱设置

### 五、视频生成组件开发

支持文字生成视频和图片生成视频两种模式。API服务商选择：
- Evolink
- KieAI
- ApiMart

### 六、落地页设计

**首屏5秒法则：** 让用户在5秒内明白产品是什么、能解决什么问题、为什么相信它。

**五个核心板块：**
1. Hero首屏区：大标题 + 副标题 + CTA按钮
2. 信任建立：客户背书、真实用户评价
3. 功能展示：3-8个核心功能配演示动图
4. 定价方案：三栏对比，提供免费试用和年付折扣
5. FAQ与底部CTA

### 七、SEO与推广

**关键词策略：**
- 新词策略：新发布的AI模型名称，竞争小、排名快
- 老词策略：成熟关键词，适合长期品牌建设
- 子页面策略：为每个模型、功能创建独立页面

**推广渠道：**
- SEO自然流量
- Google Ads（新手不建议一开始投）
- X Build in Public
- LinkedIn个人IP
- TikTok病毒视频

### 八、支付接入

推荐 Creem 平台：
- 对中国开发者友好，无需公司资质
- 支持支付宝或港卡提现
- 审核门槛低

**定价公式：** 售价是成本的2.5倍，毛利率约60%

### 九、风险提示

**低风险：** 前期投入可控，试错成本低
**中风险：** SEO见效慢，蹭模型品牌词有法律风险
**可控风险：** API按量计费需合理定价

## 关键要点

- 行动比完美更重要：先上线、再优化
- 产品差异化设计和落地页转化能力是关键
- AI编程工具已大幅降低技术门槛
- 持续推广的耐心不可或缺
